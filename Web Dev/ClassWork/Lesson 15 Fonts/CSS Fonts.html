Fonts in CSS

The font-family property in CSS tells the page which type of font to use for a specific element. Whatever font you specify must exist on the client machine or it will default to the standard. That's why it's better to stick with common fonts.

/* Below is the CSS code for the text on this page. */
p
{
     font-family:comic sans ms }
